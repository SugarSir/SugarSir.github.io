准备172.18.139.69，172.18.139.57 nginx+php
编译安装PHP：




useradd  -s /sbin/nologin  wwww

编译安装PHP：


make && make install

make && make install

pwd
/usr/local/php/etc/php-fpm.d
cp www.conf.default  www.conf


cd /usr/local/src
tar xvf nginx-1.14.2.tar.gz
cd nginx-1.14.2

编译安装Nginx
 ./configure --prefix=/usr/local/nginx  --with-pcre  --with-http_stub_status_module --with-http_ssl_module

make && make install


配置Nginx web目录：
mkdir /data/nginx/wordpress –p
chown  www.www /usr/local/nginx/ /data/nginx/ -R

 location / {
            root   /data/nginx/wordpress;
            index   index.php index.html index.htm;
            if ($http_user_agent ~ "ApacheBench|WebBench|TurnitinBot|Sogou web spider|Grid Service") {
                proxy_pass http://www.baidu.com;
                return 403;
            }
        }
 location ~ \.php$ {
            root          /data/nginx/wordpress;
            fastcgi_pass   192.168.7.104:9000;
            fastcgi_index  index.php;
            fastcgi_param  SCRIPT_FILENAME  $document_root$fastcgi_script_name;
             include        fastcgi_params;
        }

准备nfs (172.18.139.59)，实现图片资源共享
安装 nfs-utils
yum install nfs-utils -y

vim /etc/exports
/nfsdata/images * (rw no_root_squash)

客户端安装 nfs-utils （172.18.139.69，172.18.139.57）

yum install nfs-utils -y

showmount -e 172.18.139.59 (查看共享目录)

挂在共享目录
mount -t nfs 172.18.139.59:/nfsdata/images /data/nginx/wordpress/wp-content/uploads

mount -t nfs 172.18.139.59:/nfsdata/images /data/nginx/wordpress/wp-content/uploads

将该挂在写在配置文件中/etc/rc.d/rc.local 或者 /etc/fstab
vim /etc/rc.d/rc.local
mount -t nfs 172.18.139.59:/nfsdata/images /data/nginx/wordpress/wp-content/uploads
